# Smart Calculator (C++)

A command-line based smart calculator developed using C++.

## Features
- Basic operations: +, -, *, /
- Power calculation (^)
- Square root calculation
- Modulo operation (%)
- Exception handling
- History of last 5 calculations
- Clean menu-based interface

## Technologies Used
- C++
- Standard Template Library
- Exception Handling
- Struct for history storage

## Example
===== SMART CALCULATOR =====
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Power

7. Modulo
8. History

   ## How to Run

1. Clone the repository
git clone https://github.com/amiteshpaliwall/smart-calculator-cpp.git

2. Go to the project folder
cd smart-calculator-cpp

3. Compile the program
g++ smart_calculator.cpp -o calculator

4. Run the program
./calculator
